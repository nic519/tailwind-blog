在之前我会闭眼选择iOS，但是在 AI 可用性大幅提升的现在，Android阵营里的OPPO进入了我的视线，以下是一些个人在这两种手机上的使用体验对比，列举的维度仅包含自己的使用习惯，并非全面对比

|          | **iPhone**  🍎        | **OPPO** 🤖 | 
| -------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ | 
| 通话录音 | ❌ 仅支持有感的通话录音。<br/>看过DingTalk外置骨导录音方案，效果很差 | ✅ >=10 <br/>- 可以无感录制 通话、微信语音、腾讯会议 等<br />- 可以在后续生成不同格式的文字记录：纪要、TODO事件、备忘录 | 
| 翻译     | ❌ 应该可以通过第三方App付费解决 | ✅ +5 <br/>- 支持外部录音的实时声音翻译，海外旅游对话使用场景<br/>- 支持内部APP的实时声音翻译，看视频的场景       |
| 消息推送 | ✅ >=10 <br/>- 消息推送对每个APP都可靠                                           | ❌ 没有或经常漏  |
| 骚扰电话 | ❌ +1 第三方App可以标记显示骚扰、诈骗电话，但是很麻烦  | ✅  +8 <br/>- 内置号码识别，快速分辨骚扰、诈骗电话等<br />- 可以叫手机助手自动接听并录音和实时看到对话字幕  |
| NFC      | ❌ +5 <br/>- 仅能使用公交卡和银行卡 <br />- 在日本可以免费支持西瓜卡的充值和消费<br />- 其余的，只能配合手环去扩展能力了             | ✅ +10 <br/>- 可以支持一切可以支持的  |
| 照片编辑 | ❌ 第三方app付费解决   | ✅ +5 人物消除 | 
| 隐私安全 | ✅ >=10  | ❌ 不细说了 |
| 数据传输 | ✅ +10 与mac生态配合无缝  | ✅ +9 <br/>- 安装官方软件后，也可以达到无缝传输，且包含windows生态 <br/>- 扣1分是因为软件有点重 |
| 其他   | ✅ +10 <br/>- FindMy: 10块钱就可以使用定位可靠的AirTag <br />- 快捷App: 里面有很多轻量的脚本可以自行编写和复用别人的脚本                                  | ✅ +5 <br/>- 三指上滑可以记录当前页面的信息，如小红书等，识别里面的文字，变成自己的知识库 |
| **汇总** | **47** | **51** |



**结论**

OPPO 在目前阶段相对来说分值高一些。但说句心里话，如果让我仅能使用1部手机，那肯定会是 iPhone
